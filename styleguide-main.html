  • The base HTML font-size on mobile tablet laptop wide-screens is 13.5px 15.75px 18px 20.25px
  • The base HTML line-height (i.e. --baseline in my SCSS) in the 'basic-grid-typography' setup 32/18 or 1.777778rem ( 32px 28px 32px 36px in this grid).
  • A single grid row is half of 1.777778rem –– or 0.8888889rem.
  • Margins and paddings should ALWAYS be multiples of these numbers to maintain vertical rhythm.

1). We always set the font-size in pixels on the `` element and size everything else in REMs ( html{ font-size: 18px; } ) . Then we use Media queries to adjust the HTML font-size and everything else scales automatically.
